Interview Prep

Useful collection of docs that I've put together for tech interview prep.

I originally had a separate webpage dedicated to each of the topics in this section but decided to just add a link (link below) to all the resources stored in my Dropbox. Yes yes, it’s just me being lazy, but it would’ve taken way too much time creating a page for each document given that there are so many.

My intent for putting together these documents was so that I could have a study guide when it came time to prep for interviews. I didn’t slap together these documents overnight, but rather it was put together throughout many years. Whenever I came across something that I wasn’t sure of, whether it was at home or at work, I’ll add it to the appropriate document. I figured this was a good idea because I’d have everything I need in one place, so no need to Google everything again and again.

Here's the link to all my resources

  • Object Oriented Principles – why it’s useful, four main principles of OOD, abstract classes vs interfaces
  • Useful Java Stuff – JVM, important algorithms, algorithm design techniques, Big O, important keywords, shallow vs deep copy, String vs StringBuilder vs StringBuffer, hash functions and collision, Hashtable vs HashMap vs LinkedHashMap vs Hashset, Java Collection Framework, ArrayList vs LinkedList vs Vector, “=” vs “==” vs “qual(), DFS vs BFS, etc.
  • Javascript
  • Microservies Architecture– intro, API Gateway, client-service interaction styles, IPC, service discovery, deployment strategies
  • RESTful API
  • Common Node Interview Qs
  • DBMS – ACID, BASE, CAP, caches, normalization
  • Programming Paradigms
  • System Design Qs
  • and a lot more…
