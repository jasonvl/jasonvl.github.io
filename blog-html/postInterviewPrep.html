Tech Interview Prep

Collection of notes that I've put together for tech interview prep.

June 07, 2019

I originally had a separate webpage dedicated to each of the topics in this section but decided to just add a link (below) to all the resources stored in my Dropbox.

My rational for putting together these notes was so that I could have a study guide when it came time to prep for interviews. I didn’t slap together these documents overnight, but rather it was put together throughout many years. Whenever I came across something that I wasn’t sure of, whether it was at home or at work, I always made sure it got added to my notes. I figured this was a good practice because I’d have everything I need in one place, so no need to Google everything again and again.

Here's the link to all my resources

In my Dropbox, you’ll find info on:

  • Object Oriented Principles
  • DBMS – ACID, BASE, CAP, caches, normalization
  • Microservice Architecture – API Gateway, client-service interaction styles, IPC, service discovery, deployment strategies
  • Messaging Systems - Kafka
  • Web Communication Technologies - RESTful API, SOAP, gRPC
  • Data Interchange Formats - JSON, XML, Binary
  • Build Tools - Maven and Gradle
  • Testing - Junit and Mockito
  • Programming Paradigms
  • System Design Qs
  • Coding Puzzles
  • Useful Java Stuff – JVM, important algorithms, algorithm design techniques, Big O, important keywords, shallow vs deep copy, String vs StringBuilder vs StringBuffer, hash functions and collision, Hashtable vs HashMap vs LinkedHashMap vs Hashset, Java Collection Framework, ArrayList vs LinkedList vs Vector, “=” vs “==” vs “qual(), DFS vs BFS, etc.

    • Disclaimer: I did not create the original content found in the notes since it came from online resources such as Stackoverflow, Youtube, Hackernoon, Medium, Udemy, etc.
